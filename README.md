# Interactive Audio Network System

## Required softwares
1. Max MSP
2. touchOSC on your choice of hardware
3. JACK and Jack Trip

## How it works
1. Run JACK and Jack Trip
2. Run Max MSP
3. Run the python script
4. You are ready for some fun!
